# 魅龙枫叶斜转

# 中心部分

枫叶飘零-枫叶斜转魔方还原教程 这三块三循环公式() 所有的字母都是转60度, 「转」字表示中心块180 顺: (R’转R L’转L)2 逆: (L’转L R’转R)2

然后还有一组公式, 换的是图中白色对称位的(其实自己根据上面一组想想也能推出来) 顺: (R转R’ L转L’)2 逆: (L转L’ R转R’)2

建议先不要用公式, 先用直觉换到差不多, 很难再推进了, 再开始用公式. 如果最后只剩下两个叶子块互换, 那就用setup制造一个三循环, 最后reverse
